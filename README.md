# CareerKick Website

Official corporate website for CareerKick Services.

This project is a modern high-performance single page application focused on delivering a premium storytelling experience for institutional audiences.

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- GSAP

## Features

- Fully responsive modern UI
- Cinematic scroll-based animations
- Modular component architecture
- Optimized performance and lazy loading
- Scalable design system
- Accessible interaction patterns
